
# 今日头条自动化脚本 - 功能总结

## 项目概述

TAUREN-SCRIPT 是一个基于 Python 的今日头条自动化脚本，支持批量点赞、评论等自动化操作，提供 Web 控制面板进行可视化管理。

## 核心功能

### 1. 内容自动化处理

#### 推荐内容处理
- **文件**: [toutiao/toutiao.py](toutiao/toutiao.py)
- **功能**:
  - 自动获取首页推荐内容
  - 支持配置每次处理的推荐内容数量（max_links_per_cycle）
  - 自动点赞推荐内容
  - 自动评论推荐内容

#### 关注用户处理
- **文件**: [toutiao/toutiao.py](toutiao/toutiao.py)
- **功能**:
  - 自动获取关注用户列表
  - 支持配置每次处理的用户数量（users_per_cycle）
  - 支持配置每个用户处理的视频数量（videos_per_user）
  - 循环处理所有关注用户

#### 内容点赞
- **文件**: [operation/operation_toutiao.py](operation/operation_toutiao.py)
- **功能**:
  - 支持视频页面点赞
  - 支持文章页面点赞
  - 多种定位器策略，提高成功率
  - 检测已点赞状态，避免重复点赞

#### 智能评论
- **文件**: [operation/operation_toutiao.py](operation/operation_toutiao.py)
- **功能**:
  - 支持视频页面评论
  - 支持文章页面评论
  - 关键词匹配生成智能评论
  - 直接输入文本评论
  - 多种定位器策略

### 2. Web 控制面板

#### 实时监控
- **文件**: [web/app.py](web/app.py)
- **功能**:
  - 实时显示脚本运行状态
  - 显示当前循环次数
  - 显示启动时间
  - 运行状态指示器

#### 数据统计
- **文件**: [web/app.py](web/app.py)
- **功能**:
  - 总处理链接数统计
  - 成功点赞数统计
  - 失败点赞数统计
  - 成功率计算与显示

#### 一键控制
- **文件**: [web/app.py](web/app.py)
- **功能**:
  - 启动脚本
  - 停止脚本
  - 后台线程运行

#### 在线配置
- **文件**: [web/app.py](web/app.py), [web/templates/dashboard.html](web/templates/dashboard.html)
- **功能**:
  - 可视化编辑所有配置参数
  - 基础配置（延迟、重试次数等）
  - 浏览器配置（无头模式、窗口大小）
  - 循环配置（循环次数、间隔、处理数量）
  - 日志配置（日志级别、目录）
  - 配置保存与加载

#### 账号管理
- **文件**: [utils/account_manager.py](utils/account_manager.py)
- **功能**:
  - 添加账号
  - 编辑账号
  - 删除账号
  - 账号状态监控
  - Cookie 文件管理
  - 操作次数统计
  - 最后使用时间记录

#### 实时日志
- **文件**: [web/app.py](web/app.py)
- **功能**:
  - 实时显示脚本运行日志
  - 时间戳显示
  - 日志级别区分（INFO/ERROR/WARNING）
  - 最多保留 100 条日志

- **文件**: [utils/config.py](utils/config.py)

- **功能**:

- **文件**: [load/load.py](load/load.py)

- **文件**: [main.py](main.py)

  - 资源清理

## 技术架构

### 后端技术
- **Flask**: Web 框架，提供 RESTful API
- **Helium**: 高级网页自动化库
- **Selenium**: 底层浏览器自动化
- **EdgeDriver**: Edge 浏览器驱动

### 前端技术
- **HTML5**: 页面结构
- **CSS3**: 样式设计
- **JavaScript**: 交互逻辑
- **响应式设计**: 适配不同屏幕

### 数据存储
- **YAML**: 配置文件存储
- **JSON**: 账号数据存储
- **Pickle**: Cookie 缓存
- **文件系统**: 日志文件

## 配置参数

| 配置项 | 说明 | 默认值 |
|---------|------|---------|
| operation.min_delay | 最小延迟(秒) | 2 |
| operation.max_delay | 最大延迟(秒) | 4 |
| operation.max_retries | 最大重试次数 | 2 |
| operation.max_links_per_cycle | 每次处理推荐内容数 | 10 |
| operation.cycle.max_cycles | 最大循环次数 | 5 |
| operation.cycle.cycle_interval | 循环间隔(秒) | 3600 |
| operation.cycle.users_per_cycle | 每次处理用户数 | 10 |
| operation.cycle.videos_per_user | 每个用户处理视频数 | 2 |
| operation.browser.headless | 无头模式 | false |
| operation.browser.maximize | 最大化窗口 | true |
| logging.level | 日志级别 | INFO |
| logging.log_dir | 日志目录 | logs |

## API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 控制面板页面 |
| `/api/status` | GET | 获取脚本状态 |
| `/api/start` | POST | 启动脚本 |
| `/api/stop` | POST | 停止脚本 |
| `/api/config` | GET | 获取配置 |
| `/api/config` | POST | 更新配置 |
| `/api/accounts` | GET | 获取账号列表 |
| `/api/accounts` | POST | 添加账号 |
| `/api/accounts/&lt;id&gt;` | PUT | 更新账号 |
| `/api/accounts/&lt;id&gt;` | DELETE | 删除账号 |


```bash
python main.py
```
